
# Apple Landing Page Clone

A responsive replica of the Apple landing page built with modern web technologies.

## Features

- Clean, minimalist design matching Apple's aesthetic
- Fully responsive layout for desktop, tablet, and mobile
- Smooth animations and transitions
- Product showcase sections
- Call-to-action buttons

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository
2. Open `index.html` in your browser

## Project Structure

```
apple-mock/
├── index.html
├── styles/
│   └── main.css
├── scripts/
│   └── main.js
└── assets/
    └── images/
```

## Usage

Simply open the HTML file or serve it using a local development server.
